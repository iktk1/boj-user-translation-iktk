0으로 만들기

문제

$N \times N$ 크기의, $0$과 $1$로만 이루어진 행렬이 주어진다. 당신은 이 행렬에 아래의 연산을 $0$회 이상 수행할 수 있다.

  • $1$ 이상 $N$ 이하의 정수 $R$과 $C$를 고르고, $1 \le r \le R$과 $1 \le c \le C$를 만족하는 모든 정수 순서쌍 $(r, c)$에 대응하는 칸을 반전시킨다. 반전시킨다는 것은 그 칸의 값이 $0$이었다면 $1$로, $1$이었다면 $0$으로 바꾸는 것을 의미한다.

행렬의 모든 칸의 값을 $0$으로 만들기 위한 연산의 최소 횟수를 구하여라.

입력

첫째 줄에 $N$이 주어진다. ($1 \le N \le 10$)

이후 $N$개 줄에 걸쳐, 행렬의 정보가 주어진다. 행렬의 $i$번째 줄에는 $i$번째 행의 칸들의 값이 붙어서 주어진다. 각 칸의 값은 항상 0 또는 1이다.

출력

문제의 정답을 출력한다.
